# Mirror Labs WebApp - Phase 1

Web-based 3D collaboration platform for construction and real estate.

## Tech Stack
- React 18 + TypeScript + Vite
- shadcn/ui + Tailwind CSS
- React Router

## Development
```bash
npm install
npm run dev
```

## Pages
- Landing (/)
- Product (/product)
- Use Cases (/use-cases)
- Contact (/contact)
- Demo (/demo)
- My Projects (/projects) - Mock data
- Project Detail (/projects/:id) - Mock data

## Phase 1 Note
Visual reference only. No auth, no backend, no 3D viewer yet.
